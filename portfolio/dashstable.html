DashsTable

Built using: HTML, CSS, and JavaScript

Status: Complete

I created DashsTable for a class assignment in Fall 2023. I created it as a tool for searching restaurants nearby based on distance, rating, best match, and review count. It has integrated the Yelp API and GoogleMaps API to allow for accurate restaurant results and the ability to choose a point on the map as the search area. You can create an account on the website and all the information is stored into a database so you can login at a later date.
